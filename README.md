# rickroll


**Informationen**
- Beschreibung: Trolle andere indem du sie durch deinen "angeblichen" Online Merchandise Shop rickrollst.
- Dauer: ca. 5-10 Minuten
- Schwierigkeit: 1/5
- Benotigt: Domain (oder Subdomain)
- YouTube Video: https://youtu.be/
- Kontakt (bei Fragen): contact@letssukitv.de


**Tutorial**
1. Gehe auf die Seite des Rickroll-Site-Generators* und erstelle dir eine Rickroll Seite. (wichtig: stelle ein, dass die Seite permanent aktiv bleibt)
2. Erstelle eine Subdomain von deiner Website.
3. Erstelle eine Datei in den Dateien deiner Subdomain bei *Dateien > {subdomain} > httpdocs >* und nenne diese *index.php*.
4. Falls sich im *httpdocs* Ordner zusätzlich eine mit dem Namen *index.html* befindet, lösche diese. (*favicon.ico* darf nicht gelöscht werden!)
5. Füge den Index Code* in die *index.php* datei ein und setze (an den gekennzeichneten Stellen) in Zeile 5 und in Zeile 14 das gewünschte Passwort / den Rickroll Link von deiner Seite ein.
6. Speichere die Dateien und dein Projekt ist fertig.
*den Zugang findest du in den Assets unter diesem Text.
